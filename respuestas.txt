Preguntas en el CSS:

1. ¿Qué propiedad CSS se usa para cambiar el color de fondo de un elemento?
   - Respuesta**: La propiedad CSS utilizada para cambiar el color de fondo de un elemento es `background-color`.

2. ¿Cómo se centra un div dentro de su contenedor padre horizontalmente?
   - Respuesta: Para centrar un `div` dentro de su contenedor padre horizontalmente, se utiliza `margin: 0 auto;`. 
   Esto establece un margen automático a ambos lados del `div`, lo que lo centra horizontalmente.

3. Cuál es la diferencia entre 'id' y 'class' en CSS?
   - Respuesta**: La principal diferencia entre `id` y `class` es que:
     - `id`: Debe ser único en la página, es decir, solo puede aplicarse a un único elemento.
     - `class`: Puede ser reutilizado en varios elementos para aplicar estilos comunes a ellos.
     
 Preguntas en el JavaScript:

1. ¿Cuál es la función de 'document.getElementById' en JavaScript?
   - **Respuesta**: `document.getElementById` es una función en JavaScript que se utiliza para obtener un elemento del DOM (Document Object Model) a partir de su atributo `id`. Esto permite manipular o interactuar con dicho elemento.

2. ¿Qué hace 'prompt' y cómo se puede usar en lugar de 'alert'?
   - **Respuesta**: `prompt` es una función que muestra un cuadro de diálogo con un campo de entrada para el usuario. A diferencia de `alert`, que solo muestra un mensaje, `prompt` permite al usuario introducir texto, que luego se puede almacenar en una variable. Se utiliza para obtener datos del usuario en lugar de solo mostrar información.

3.¿Cuál es el propósito de la instrucción 'if' en este fragmento de código?
   - **Respuesta**: La instrucción `if` se utiliza para evaluar una condición, en este caso, si la respuesta introducida por el usuario es igual a `'paris'`. Si la condición es verdadera, se ejecuta el bloque de código que muestra el mensaje "Correct!". Si la condición es falsa, se ejecuta el bloque `else` que muestra el mensaje "Try again!".

4.¿Cómo se puede manipular el DOM usando JavaScript para cambiar el contenido de un elemento?
   -Respuesta: Para manipular el DOM y cambiar el contenido de un elemento, se utiliza la propiedad `innerText` o `innerHTML` de un elemento obtenido, por ejemplo, con `document.getElementById`. En el código dado, `document.getElementById("question").innerText = "What is the capital of Spain?";` cambia el texto del elemento con id `question` a "What is the capital of Spain?".
